# Brand logo

The app reads the logo from these files. Replace them with your official
files (keep the same names, transparent background) — no code changes needed:

- `public/logo.svg` → color mark, used on light backgrounds
- `public/logo-light.svg` → white mark, used on dark / navy backgrounds

The current files are placeholders that approximate the Auto Palestine mark.
SVG is recommended (crisp at any size). PNG also works if you update the
extension in `src/components/common/logo.tsx`.
